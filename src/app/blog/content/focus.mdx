### **How to start**

- Count down 5 seconds and start ([The 5 second rule](https://en.wikipedia.org/wiki/Five-second_rule))
- Start with the first 5 min of work (you'll probably do it longer)
- Use a Promodoro Timer for difficult tasks ([Pomodoro Technique](https://en.wikipedia.org/wiki/Pomodoro_Technique))
