## 1. One source of truth

Every agent has its own config file:

- Codex reads `AGENTS.md`
- Claude reads `CLAUDE.md`
- Cursor has rules

Copy/paste them and they drift.

Keep one source file instead. Symlink it into each agent.

```bash
ln -s ~/.agents/system_prompt.md ~/.codex/AGENTS.md
ln -s ~/.agents/system_prompt.md ~/.claude/CLAUDE.md
```

Now updates happen once.

## 2. Writing style

Add a `writing_style.md`.

Agents can write, but they drift into generic AI voice.

Too polished. Too long. Not how I would write.

The file should include concrete examples from your own:

- messages
- comments
- docs
- replies

My version includes rules like:

- English in team channels, German in German 1:1s
- Use short paragraphs
- Make direct asks
- Use `Context:` and `Impact:` when useful
- Avoid "I hope this helps"

Tell agents to read this file before drafting anything in your name.

Still review the output. The goal is less rewriting, not zero review.

## 3. Skills

Baseline instructions define agent behavior.

Skills define sharper opinions for specific work.

Examples:

- [`typescript-principles`](https://github.com/mrzmyr/agent-plugin/blob/main/skills/typescript-principles/SKILL.md)
- [`deslop`](https://github.com/mrzmyr/agent-plugin/blob/main/skills/deslop/SKILL.md)

`typescript-principles` covers TypeScript defaults:

- Prefer named parameters over positional parameters
- Prefix boolean functions with `is`, `has`, `can`, `should`, etc.
- Use short action-based function names
- Write structured errors with `status`, `message`, `why`, and `fix`

`deslop` pushes against lazy agent code:

- Question fallbacks
- Question regex parsing when a real parser exists
- Use popular libraries for solved problems
- Avoid type assertions when inference should work

The goal: make taste executable.

## 4. File structure

```txt
~/.agents/
  system_prompt.md
  writing_style.md
  skills/
    typescript-principles/
      SKILL.md
    deslop/
      SKILL.md

~/.codex/AGENTS.md -> ~/.agents/system_prompt.md
~/.claude/CLAUDE.md -> ~/.agents/system_prompt.md
```

The system prompt sets defaults.

`writing_style.md` keeps communication close to my voice.

Skills add pressure for the work at hand.

## 5. Review

This setup does not make agents perfect.

It removes drift.

That is usually enough.
