## Skills

- See [skills repository](https://github.com/mrzmyr/agent-plugin/blob/main/skills).

## Symlink Context

- Add `~/.agents/system_prompt.md`.
- Add symlinks to `~/.codex/AGENTS.md`, `~/.claude/CLAUDE.md`, etc.

## `.agents/`

- `.agents/` is a remote git repository
- Updated by hooks when you commit to `~/.agents/`

## Writing style

`writing_style.md` is used whenever agents write in your name.

- Create `~/.agents/writing_style.md` from existing messages
- Use slack, emails, PR comments, docs, replies.
- Reference in `~/.agents/system_prompt.md`:

```md
# Writing style

## Rules

- English in team channels, German in German 1:1s
- short paragraphs
- direct asks
- use `Context:` and `Impact:` when useful
- avoid "I hope this helps"

## Examples

- can you add @Name to the cursor team?
- used whenever agents write in your name
```
